```text

```

## v8.3.1の主な変更

```text
仲間画面をキャンプ風のゲーム画面へ刷新
現在の先頭モンスターとパーティ状況を大きく表示
パーティを専用編成カードと3枠レールで表示
牧場カードに通常加入・一括交換ボタンを常時表示
最後の1体を牧場へ戻せない理由を画面内に表示
仲間検索を折りたたみ式パネルへ整理
名前検索中のフォーカス維持
PC・スマホ表示を調整
交換処理・セーブ形式は変更なし
```
