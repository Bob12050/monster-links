# モンスターリンクス v8.4.1

スマホでも遊べる、ソロ向けモンスター育成・配合RPGです。

## 現在の安定版

```text
v8.4.1 通常攻撃オート戦闘
```

## v8.4.1の主な変更

```text
戦闘画面に通常攻撃オートを追加
プレイヤーターンごとに通常攻撃を自動選択
特技・スカウト・防御・交代・逃走で即時解除
HP25%以下、戦闘不能、敵撃破、勝敗時に安全停止
通常戦・ボス戦・闘技場開始時は必ずOFF
闘技場の次ラウンドでもOFF
古い予約タイマーと二重行動を防止
オート状態はセーブデータへ保存しない
戦闘計算・敵AI・スカウト判断は変更なし
```

## ゲーム概要

```text
敵を倒す

装備で強化する
図鑑から育成目標を決める
素材を集めて配合する
ボスを倒して次のステージへ進む
```
