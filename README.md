# モンスターリンクス v8.5.1

スマホでも遊べる、ソロ向けモンスター育成・配合RPGです。

## 現在の安定版

```text
v8.5.1 冒険ワールドマップ
```

## v8.5.1の主な変更

```text
13地域を大陸上に配置したワールドマップを追加
解放・ボス挑戦可・制覇・未解放をピンの色で表示
選択した1地域だけを詳しく表示する構成へ変更
敵、報酬、属性、ボス進行、攻略ヒントを詳細パネルへ集約
最新解放地域を初期選択
スマホでは地図を横スクロール表示
選択地点が画面中央付近へ自動移動
戦闘開始・解放条件・報酬・セーブ形式は変更なし
```

## ゲーム概要

```text
敵を倒す
仲間をスカウトする
装備で強化する
図鑑から育成目標を決める
素材を集めて配合する
任務を達成して報酬を受け取る
ボスを倒して次のステージへ進む
```
