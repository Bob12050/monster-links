# モンスターリンクス v8.3.2

スマホでも遊べる、ソロ向けモンスター育成・配合RPGです。

## 現在の安定版

```text
v8.3.2 配合目標・お気に入り
```

## v8.3.2の主な変更

```text
図鑑から欲しいモンスターを配合目標に登録
最大3体の目標を保存
拠点に最優先目標を表示
配合所に素材進捗を一覧表示
素材所持数・必要数・最高Lv・保護数を表示
素材不足・育成条件・作成可能を色分け
作成可能な目標から親2体を自動セット
目標の優先順位変更と解除に対応
未発見素材のネタバレを防止
旧セーブには空の目標情報を自動補完
```

## ゲーム概要

