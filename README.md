## 友達に遊んでもらう時

GitHub Pagesの公開URLを送れば、ブラウザから遊べます。

```text
別端末で遊ぶとセーブデータは別
ブラウザのデータを消すとセーブも消える可能性あり
まだ開発中なのでバグがあればスクショで確認
```

```text

```
