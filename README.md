# モンスターリンクス v8.2.1

スマホでも遊べる、ソロ向けモンスター育成・配合RPGです。

## 現在の安定版

```text
v8.2.1 戦闘画面・演出強化
```

## v8.2.1の主な変更

```text
敵と味方が対面する戦場レイアウトへ刷新
HP・MP・ランク・属性を戦闘HUDに整理
控えパーティと残りHPを戦闘中に表示
最新行動・属性相性・行動待機状態を常時表示
攻撃・特技・スカウト・防御・交代・逃走コマンドを再設計
特技選択と交代モーダルを画像・相性情報付きに強化
攻撃・被ダメージ・回復・防御・撃破演出を強化
撃破と戦闘不能の演出後に画面遷移するようテンポを調整
戦闘計算・敵AI・報酬・セーブ形式は変更なし
```

## ゲーム概要

```text

装備で強化する
配合で新しい仲間を作る
ボスを倒して次のステージへ進む
```
