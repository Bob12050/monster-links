# v8.0 確認チェックリスト

## パーティ枠

```text
1枠モンスター3体で 3/3枠 になる
2枠+1枠で 3/3枠 になる
2枠+2枠は編成できない
3枠モンスターは単独編成になる
枠不足の牧場モンスターはパーティへボタンが無効になる
```

## スカウト

```text
枠が空いていればパーティへ入る
枠が足りなければ牧場へ送られる
報酬画面に加入先が表示される
```

## 配合

```text
配合後の子はLv1
引き継ぎ技は最大2つ
枠が空いていればパーティへ入る
枠が足りなければ牧場へ送られる
```

## 表示

```text
仲間一覧の枠ゲージが正しく表示される
詳細モーダルに必要枠・残り枠が出る
図鑑で1枠/2枠/3枠フィルターが使える
仲間一覧で1枠/2枠/3枠フィルターが使える
戦闘画面に敵味方のサイズバッジが出る
```

## セーブ

```text
旧セーブを読み込んでも起動する
枠超過パーティは超過分が牧場へ移る
バックアップ復元後も枠超過しない
```
