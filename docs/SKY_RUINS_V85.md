# v8.5 天空遺跡

## 追加エリア

`魔界門`のボスを倒す、またはスカウトすると第13エリア`天空遺跡`が解放される。

```text
推奨Lv: 68
通常敵Lv: 68〜82
ボスLv: 86
通常探索7勝でボス挑戦
```

光、翼、機械、高速をテーマにした終盤エリア。魔界門より敵の素早さと賢さが高く、回復役と耐久装備を含めた編成を想定している。

v8.5.2で実戦シミュレーションを行い、通常探索はLv68、ボス戦はLv72前後または天空配合の仲間を目安に調整した。

## 新モンスター

```text
通常出現
雲ぷる / サンラビ / ゲイルグリフ / スカイウォーデン / ストームジン

配合育成
エーテルゴーレム / セラファルコン / ヘヴンスケイル

ボス
ゼニスドラゴン
```

新規モンスターは既存スキルだけで構成しているため、戦闘処理とセーブ形式への変更はない。

## 新配合

10件の固定配合を追加した。天空遺跡でスカウトできるモンスターと既存の終盤モンスターを組み合わせ、最終的に`ゼニスドラゴン`へ到達する。

ゼニスドラゴンには次の2ルートがある。

```text
ヘヴンスケイル + セレスティアルセラフ
ソーラーウィルム + セレスティアルセラフ
```

## 新装備

```text
天空晶片: MP・素早さ・賢さ
蒼天の羽: HP・MP・素早さ・賢さ
天頂竜の核: 全能力とHP・MP
```

通常探索、上位モンスターのドロップ、ボス任務報酬へ分散している。

## 新任務

```text
天空遺跡の天頂竜
天空の図鑑博士
十三の守護者を越えて
```

## アート

新規13点はGitHub Pagesで直接表示できるSVG仮アート。外部サーバーや画像ビルドは不要。

正式イラストへ更新する場合も、データ内の`image`参照を同じファイル名のPNGまたはSVGへ差し替えればよい。

## セーブ互換性

保存形式番号は`1`のまま。既存セーブ読み込み時に`stageWins.sky_ruins`と`bossCleared.sky_ruins`が自動補完される。

既存モンスター、装備、任務IDは変更していない。
