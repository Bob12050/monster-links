# v6.4 図鑑・仲間一覧の検索/フィルター強化

## 今回の目的

今後モンスター数を150匹規模に増やす前に、図鑑と仲間一覧を探しやすくしました。

## 図鑑に追加した機能

```text
名前検索
ランク絞り込み
属性絞り込み
状態絞り込み
表示中の件数表示
リセットボタン
```

## 図鑑の状態フィルター

```text

```
