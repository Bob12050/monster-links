## 監査結果

主要機能と画面構成はすでに揃っています。
今回の最大の問題は、情報不足ではなく説明文字の小ささでした。

```text
CSS内の6〜9px指定: 119か所
対象: 戦闘、仲間、図鑑、配合、任務、ワールドマップ
```

## 今回の改善

```text
上部バーへ現在の画面名を追加
下部ナビへaria-currentを追加
操作要素へキーボードフォーカス表示を追加
主要な補助文字を11px前後へ拡大
スマホ時も重要情報を10px未満にしない
小型ボタンの高さを42px以上へ調整
```
