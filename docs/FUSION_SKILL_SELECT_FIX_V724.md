# v7.2.4 引き継ぎ技を2つ選べない不具合修正

## 原因

配合プレビューを描画するたびに、引き継ぎ技の選択状態を自動補正していました。

そのため、技を外して選び直そうとしても、再描画時に初期技がまた自動選択され、好きな2つを選びにくい状態になっていました。

## 修正内容

```text
親2体を選んだ時だけ初期技を最大2つ自動選択
プレビュー描画時には技選択を勝手に初期化しない
技のクリック時は現在の選択状態を維持
0〜2個の範囲で選び直せるように修正
配合実行直前にだけ選択技を安全確認
```

## 仕様

```text
親を選ぶ: 初期候補から最大2つ自動選択
技をタップ: ON/OFF切り替え
最大: 2つ
0個のまま配合した場合: 実行時に候補から自動補完
```
