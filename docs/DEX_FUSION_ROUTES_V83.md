# v8.3 図鑑配合ルート

## 目的

図鑑で欲しいモンスターを見つけ、必要素材を確認し、配合画面へ進む流れを一つにつなげる。

## 図鑑詳細

発見済みの図鑑カードをタップまたはクリックすると詳細を開く。

- モンスター画像
- ランク・属性・サイズ
- 習得技
- 発見・スカウト状態
- 現在の所持数
- 作成ルート数
- 素材として使うルート数

## 作成ルート

対象モンスターが結果になる固定配合を表示する。

- 親2体
- 未保護素材の所持数
- 保護中の素材数
- 素材の最高Lv
- レベル・ランク・サイズ条件
- 作成可能 / 条件未達 / 素材不足

素材と条件を満たし、仲間が3体以上いる場合だけ「この配合をセット」を使用できる。

## 素材逆引き

対象モンスターを親として使う固定配合を表示する。配合結果が発見済みなら名前と画像を表示する。

## ネタバレ制御

- 未発見の親は「？？？？」で表示
- 未発見の結果は名前と画像を非表示
- 未発見または実行不能なルートのボタンにレシピキーを出力しない
- 未発見結果への直接セットは無効

## 互換性

既存の図鑑発見記録、所持モンスター、配合レシピ、配合条件を参照する表示機能である。
セーブキー、保存項目、保存形式、配合結果は変更していない。
