# v7.1.5 モンスター保護ロック

## 目的

配合が本格化してきたため、大事なモンスターを誤って配合素材にする事故を防ぐための保護ロックを追加しました。

## 追加内容

```text
仲間カードに「保護」ボタンを追加
保護中の仲間に 🔒 保護中 バッジを表示
保護中の仲間は配合素材に選べない
おすすめ配合・配合リストの素材候補から保護中の仲間を除外
配合実行時にも保護中なら停止
```

## 使い方

```text
仲間
↓
保護したいモンスターの「保護」ボタン
↓
🔒 保護中 が表示される
```

解除したい場合は「保護解除」を押します。

## 注意

```text
保護ロックは配合素材化を防ぐ機能です
パーティ移動・装備変更は通常通りできます
```
