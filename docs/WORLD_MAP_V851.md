# v8.5.1 冒険ワールドマップ

## 目的

従来のステージカード一覧を、大陸を進んでいることが分かるワールドマップへ変更した。

地域の情報量や戦闘導線は維持しつつ、現在地、制覇した地域、次の目的地を見つけやすくしている。

## 地図

13地域を曲線ルートで接続している。

```text
青: 解放済み
赤: ボス挑戦可能
金: ボス制覇済み
灰: 未解放
黄の光: 現在選択中
```

未解放地域も選択できるが、通常探索とボス挑戦は実行できない。解放条件を詳細パネルで確認できる。

## 地域詳細

選択中の1地域について次の情報を表示する。

```text
推奨Lv
敵レベル
危険度
ボス出現までの探索進行
属性傾向
出現モンスター
主な報酬
エリアボス
攻略ヒント
```

通常探索とボス挑戦は従来の処理を呼び出すため、戦闘計算と報酬は変更していない。

## スマホ対応

地図部分だけを横スクロール可能にし、地域詳細は画面幅に合わせて縦表示する。

画面を開いた時と地点を選択した時に、選択ピンが地図中央付近へ自動移動する。

演出軽減設定が有効な場合は、選択ピンと次地点のアニメーションを停止する。

## セーブ互換性

地図で選択中の地域は一時的な画面状態であり、セーブデータには保存しない。

既存の次の値をそのまま利用する。

```text
stageUnlocked
stageWins
bossCleared
```

保存形式番号は`1`のまま。

## 検証

次を自動検証へ追加した。

```text
ステージ数と地図地点数の一致
天空遺跡を含む全地点の生成
選択地域の詳細表示
未解放地点の表示
不正な地域IDの無視
スマホ用自動スクロール処理
```
