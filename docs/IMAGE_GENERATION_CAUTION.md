# 画像生成時の注意点

## 今回起きた問題

v4.4作業中、個別PNGを作りたい場面で、アップデート告知風の一覧画像が生成されました。

これはゲーム素材としては使いません。

## ゲーム素材に使える条件

```text
1体だけが描かれている
文字が入っていない
背景が透明
キャラクターが中央にいる
1024x1024の正方形
```

## 使わない画像

```text
アップデート告知画像
複数キャラの一覧画像
文字が入った画像
背景が焼き付いた画像
切り出しが必要な画像
```

## 今後の対応

画像生成は、1体ずつ明確に対象を指定して行います。
生成後は透過チェックをしてから `assets/images/monsters/` に入れます。
