# v7.4 ドロップ・報酬調整

## 目的

追加モンスター第1弾にドロップ報酬を設定し、通常戦闘・配合素材集め・図鑑埋めの周回に意味を持たせました。

## 方針

```text
新規アイテムは追加しない
既存アクセサリー・素材系報酬を使う
序盤〜中盤の追加モンスターは基本アクセ中心
上位配合モンスターはレアアクセを低〜中確率で設定
Sランク titanplim はレア寄りだが、強すぎる報酬にはしない
```
