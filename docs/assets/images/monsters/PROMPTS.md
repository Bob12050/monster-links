# モンスター画像プロンプト集 v2.7

## 画像ルール

- 形式: PNG推奨
- 背景: 完全透明
- 比率: 1:1 正方形
- 推奨サイズ: 512x512 または 1024x1024
- 構図: 中央に1体、全身、正面または少し斜め
- 画風: かわいいがバトルゲーム向け、丸みのあるデフォルメ、太めの輪郭、軽いセル塗り
- 禁止: 文字、ロゴ、地面の影、枠、背景、市松模様

ゲーム側は同じIDのPNGを優先表示します。例: `plim.png` を置くと `plim.svg` より優先されます。
