- 形式: PNG推奨
- 背景画像なので透過不要
- 推奨比率: 16:9 または 4:3
- 推奨サイズ: 1280x720 以上
- 文字なし、ロゴなし、キャラクターなし
- 戦闘画面の背景として使うため、中央は少し空ける
