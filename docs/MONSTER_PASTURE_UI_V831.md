# v8.3.1 仲間・牧場画面刷新

## 目的

仲間画面を、パーティの状態と牧場からの入れ替え方法が一目で分かるゲーム画面へ刷新する。

## パーティ編成

- キャンプ背景と現在の先頭モンスターを表示
- 3枠の使用状況をレールで表示
- 各メンバーのHP、MP、レベル、属性、サイズを表示
- 詳細・装備、先頭変更、牧場移動をカード上から操作
- 最後の1体は牧場へ戻せないことを画面内で説明

## 牧場管理

- モンスター画像と主要ステータスを2列のカードで表示
- 空き枠があれば「パーティに加える」を表示
- 枠不足なら「交換して加える」を表示
- 交換時は既存の一括交換モーダルを使用
- 名前、ランク、属性、サイズ、居場所による絞り込みを維持
- 名前入力による再描画後もフォーカスとカーソル位置を維持

## スマホ表示

- パーティカードを画像と情報の横並びに変更
- 牧場カードを1列表示
- キャンプの先頭モンスターを上部へ重ねて表示
- 操作ボタンを押しやすい幅で配置

## 互換性

表示と操作導線のみを変更している。
セーブキー、保存項目、モンスターデータ、パーティ枠判定、一括交換処理は変更していない。
