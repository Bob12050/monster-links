# v8.1.2 パーティ一括交換

## 目的

3枠モンスターを牧場からパーティへ直接入れられない問題を修正します。

## 操作

```text
牧場の仲間を開く
「交換してパーティへ」を押す
牧場へ戻す現在のパーティメンバーを選ぶ
交換後の使用枠と編成を確認
「選んだ仲間と交換」を押す
```

必要枠をちょうど確保しやすい組み合わせを最初から自動選択します。
選択内容は手動で変更できます。

## 安全性

```text
牧場の加入対象が存在するか確認
選択した仲間が現在のパーティにいるか確認
交換後の合計枠が上限以内か確認
すべて成立した場合だけパーティと牧場を一括更新
```

途中でパーティが0体になる状態は保存されません。
