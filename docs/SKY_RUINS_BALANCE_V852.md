# v8.5.2 天空遺跡バランス調整

## 目的

魔界門から天空遺跡へ進んだ時の難易度、報酬、スカウト、素材収集を数値で確認し、表示される推奨Lvと実際の攻略難度を近づける。

## シミュレーター

`tools/balance-sim.mjs`を追加した。

ゲーム本体と同じモンスター定義、装備、属性倍率、ダメージ倍率、ボス補正を読み込む。固定IVと性格を使い、同じ条件を再実行できる。

```text
通常探索
ボス戦
特技と回復を選ぶ戦い方
通常攻撃だけを使うオート相当
```

代表編成ごとに各1200戦を実行する。

実行方法:

```text
node tools/balance-sim.mjs
```

## 調整結果

```text
挑戦下限: Lv64 → Lv68
通常敵: Lv68〜84 → Lv68〜82
ゼニスドラゴン: Lv88 → Lv86
スカウト基礎値: 3 → 8
```

Lv68の通常探索:

```text
2体の既存回復編成
手動特技 100%
通常攻撃のみ 約72%

3体の既存終盤編成
手動特技 100%
通常攻撃のみ 約95%
```

ゼニスドラゴン:

```text
既存回復編成Lv72
調整前 約44%
調整後 約66%

天空配合編成
安定攻略可能

通常攻撃のみ
勝率0%
```

ボスは通常攻撃オートで突破する相手ではなく、特技、回復、編成準備を使う最終戦として維持している。

## ドロップ

次の素材ドロップ率を改善した。

```text
雲ぷる 天空晶片: 8% → 10%
サンラビ 天空晶片: 8% → 10%
スカイウォーデン 天空晶片: 10% → 12%
ゲイルグリフ 蒼天の羽: 5% → 8%
ストームジン 蒼天の羽: 12% → 14%
```

実際の判定では全体設定の`dropRateMultiplier`も適用される。

## スカウト

天空遺跡の基礎値を8へ変更した。

C・Bランクの新モンスターはHPを減らす、レベル差を作る、スカウトチャームを使うことで成功率を伸ばしやすくなった。Aランク以上は引き続き希少枠としている。

## セーブ互換性

ステージID、モンスターID、装備ID、任務ID、保存形式は変更していない。
