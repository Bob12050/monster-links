# v8.2.1 戦闘画面・演出強化

## 目的

戦闘ルールを変えず、敵と味方が実際に戦っている感覚、状況の読みやすさ、スマホでの操作性を強化する。

## 戦闘画面

- 敵を右上、味方を左下に配置する対面型の戦場
- 敵味方それぞれのHP、MP、ランク、属性、レベル、サイズを表示
- HP低下時にバーを黄色・赤へ変化
- 控えパーティの画像、名前、HP、戦闘不能状態を表示
- 最新行動、行動待機状態、通常攻撃の属性相性を表示

## コマンド

- 攻撃
- とくぎ
- スカウト
- 防御
- 交代
- 逃げる / 棄権

アイコン、名称、用途を同時に表示する。特技画面では属性、消費MP、敵との相性を確認できる。

## 演出

- 攻撃側の踏み込み
- 被ダメージ時の揺れとヒット画像
- 回復と防御の発光
- スカウト失敗時の動き
- 撃破時のK.O.表示
- 撃破、全滅、自動交代前の短い待機

## 互換性

ダメージ計算、属性倍率、敵AI、スカウト率、報酬、セーブ形式は変更していない。
