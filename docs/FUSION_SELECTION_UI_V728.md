# v7.2.8 配合選択UI改善

## 目的

配合画面で「誰と誰が選ばれているか分かりにくい」「選択後にわざわざ上へ戻るのが面倒」という操作上の問題を改善しました。

## 追加内容

```text
選択中の親2体を配合画面上部に表示
親1・親2・配合結果を横並びで表示
モンスター名・種族名・ランク・Lvを表示
2体選択後にプレビューへ自動スクロール
おすすめ配合から選んだ時もプレビューへ自動スクロール
配合リストからセットした時はリストを閉じてプレビューへ移動
プレビューへ移動ボタンを追加
```
