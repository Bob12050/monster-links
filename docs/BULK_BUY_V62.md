# v6.2 まとめ買い機能追加

## 今回の目的

友達レビューの「まとめ買いが欲しい」に対応しました。

## 追加内容

どうぐ屋の基本アクセサリー購入に、以下のボタンを追加しました。

```text
1個
5個
10個
買えるだけ
```

## 対象

```text
ちからのリング
まもりの石
はやての羽
まどうの首飾り
命のしずく
マナシェル
```

## 仕様

```text
所持数を表示
1個あたりの価格を表示
現在のGOLDで買える数を表示
GOLD不足時は購入ボタンを無効化
5個/10個を押しても、所持GOLDで買える数まで自動調整
買えるだけは現在GOLDで買える最大数を購入
```
