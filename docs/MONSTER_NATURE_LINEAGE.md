# Monster Links 自然系整理資料

更新日: 2026-06-08  
対象バージョン: v8.6-A.1  
資料種別: アート・系統整理メモ  
ゲーム反映状態: v8.6-A.2で既存ID分のみ反映

この文書は、Monster Links の「自然系」モンスターの系統、名称、アート方針、現在のデザイン承認状況を整理するための資料です。

現時点では、ゲームデータ・能力・配合・セーブ形式・画像ファイルへの正式反映は行っていません。  
ここでの「採用候補」「承認」は、白背景確認用デザインの方向性承認を意味します。  
透明PNG化、画像ファイル差し替え、表示名変更、ゲーム実装は、ユーザー確認後の別作業として扱います。

---

## 1. 基本方針

自然系は、森・草原・湿地・水辺に住むモンスター群として扱います。

共通方針:

```text
葉、花、苔、キノコ、木の実、蔓、根、樹皮など自然モチーフを使う
かわいさと野生感を両立する
低ランクでも弱々しくしすぎない
序盤から中盤の世界観を支える親しみやすい系統にする
守護神・王様・豪華装飾に寄せすぎない
スマホ画面でも輪郭と特徴が分かる
植物だけでなく、小動物・鳥・湿地生物・森の獣との交差も許可する
```

自然系は、ぷる系と並んで序盤の印象を作る重要な系統です。  
特に `leafling` は、自然系の基本種として扱います。

## 3. 重要な名称変更方針

### モスキング → コケヌシ

`mossking` は、名前に「キング」が入ることで王様・冠・守護神寄りに引っ張られやすかったため、表示名を **コケヌシ** にする方針です。

```text
既存ID: mossking
新表示名候補: コケヌシ
```

IDは配合・図鑑・セーブ互換性に関わるため、原則として `mossking` のまま維持します。  
変更する場合は、表示名だけを変更するのが安全です。

コケヌシの方向性:

```text
最初の森ステージの自然系ボス
森の床に住む苔のぬし
低く、横に広く、どっしりした体型
苔むした背中
淡い樹皮の顔
葉っぱの眉
緑の目
太い根・切り株の手足
木の爪のような手
小さなキノコ、クローバー、小花、芽
王冠・杖・宝石などの豪華装飾は避ける
```

## 4. 自然系の中核ライン

### 4.1 リーフリン系

```text
リーフリン
```

自然系の基本種です。  
ぷる系でいう「ぷるミン」に近い入口枠として扱います。

デザイン方針:

```text
葉っぱ・新芽・小さな植物体
明るい緑と黄緑
丸く親しみやすい顔
小型で分かりやすいシルエット
序盤モンスターとしてかわいいが、安っぽくしない
```

---

### 4.2 つぼみラビ系

```text
つぼみラビ
└─ サンラビ
```

小動物と植物を混ぜた自然系の枝です。

つぼみラビ:

```text
クリーム色の小型うさぎ
大きな未開花のつぼみ耳
葉っぱの襟・足元の葉
森の小動物感
かわいい序盤モンスター
```

サンラビ:

```text
つぼみラビの花・光派生
耳のつぼみが開いた花になる
白、黄色、黄緑の明るい配色
太陽・花・光の印象
つぼみラビより少し上位感
```

---

### 4.3 コケヌシ系

```text
コケヌシ
```

最初の森ステージのボス枠です。  
王様ではなく、「森に長く住む苔のぬし」として扱います。

デザイン方針:

```text
低くどっしりした体型
苔の丘・森の床のような体
淡い樹皮の顔
根や切り株の太い手足
葉、苔、小花、小キノコ、クローバー
序盤ボスらしく、通常モンスターより少し強そう
豪華な冠や杖はなし
```

---

### 4.4 毒森・キノコ系

```text
ドクキノコ
```

自然系の毒・湿地寄りの枝です。

デザイン方針:

```text
紫や赤紫のキノコ傘
斑点
毒胞子
緑の毒ガスやぬめり
いたずらっぽい目
怖すぎず、きもかわいい
```

## 8. 実装時の注意

この資料は設計メモです。実装時は以下を守ります。

```text
確認済みデザインのみ透過PNG化する
未承認アートをゲームに正式反映しない
画像更新と能力・配合・セーブ形式変更を混ぜない
既存IDを安易に変更しない
表示名変更だけで済む場合はIDを維持する
画像は assets/images/monsters/{id}.png に配置する
旧画像は必要に応じて legacy フォルダへ退避する
実装後は tools/validate.mjs と tools/art-audit.mjs を実行する
```

注意:

```text
mossking は表示名をコケヌシに変更しても、IDは原則 mossking のまま維持する
gearslime と同様、ID変更は配合・図鑑・セーブ互換性への影響があるため慎重に扱う
```
