# v7.8 モンスター詳細モーダル・カード省略表示

## 目的

仲間ページのスクロール量が多くなっていたため、モンストやパズドラのように、一覧はコンパクトにして詳細はタップ/クリックで開く形に変更しました。

## 今回の変更

```text
仲間一覧カードをコンパクト化
パーティ/牧場のモンスターカードをタップ/クリック可能に変更
詳細モーダルを追加
詳細モーダルにステータス・性格・個体値・装備・技を集約
詳細モーダルからパーティ移動・牧場移動・先頭変更・装備変更・保護ロックを操作可能
```
