# v8.0 パーティ枠制限本実装

## 編成ルール

```text
パーティ上限は合計3枠。
1枠モンスターは3体まで編成可能。
2枠モンスターは1枠モンスターと組み合わせ可能。
3枠モンスターは単独編成になる。
```

## 実装範囲

```text
牧場 → パーティ移動時に枠チェック
枠不足時は戻す仲間を選ぶ一括交換
スカウト成功時に枠チェック
配合結果加入時に枠チェック
セーブ読み込み時に枠超過を自動補正
仲間一覧に枠ゲージを表示
詳細モーダルに必要枠・残り枠を表示
図鑑・仲間一覧にサイズフィルターを追加
```

## 枠不足時の挙動

```text
牧場から移動：空きがあれば加入、枠不足なら一括交換画面を表示
スカウト成功：仲間にはなるが牧場へ送る
配合成功：子は生まれるが牧場へ送る
復元/旧セーブ：先頭から上限内だけ残し、超過分は牧場へ移動
```

## v8.0で大型扱いにした主な候補

```text
2枠：モスキング、オアロード、タイダルセラフ、ヴォルカザード、フロストリヴァイア、アークオートマタ、アストラルワーム、ギガントコア、フォージゴーレム、タイタンぷる など
3枠：ヴォイドドラゴン、プリズムドラゴン、オメガフレーム、アビスリヴァイア、カオスワーム、デモンロード など
```

サイズによる行動回数・耐性・スカウト率補正はまだ未実装。
大型専用配合・4体配合は将来実装予定。
