# v7.1 スカウト減衰仕様

## 目的

スカウトを連打すれば簡単に仲間にできる状態を抑えます。

## 仕様

```text
1回目: 通常成功率
2回目: 成功率 -10%
3回目: 成功率 -20%
4回目: 成功率 -30%
5回目以降: その戦闘ではスカウト不可
```

内部的には、失敗回数が4回になった時点で敵が完全に警戒します。

## 画面表示

```text
成功率 xx% / 警戒 0/4
成功率 xx% / 警戒 1/4
成功率 xx% / 警戒 2/4
成功率 xx% / 警戒 3/4
警戒中 / 不可
```

## 注意

闘技場では従来通りスカウト不可です。
