# v7.2.3 配合リスト選択時の結果ズレ修正

## 原因

配合リストからレシピを選んだ後、選択された親2体から再度レシピ判定を行っていたため、表示していたレシピ結果と選択後プレビューがズレる可能性がありました。

## 修正内容

```text
配合リストから選んだレシピキーを保持
選択後プレビューではそのレシピ結果を優先
配合実行時も同じレシピ結果を使用
手動で親を選び直した場合は固定レシピを解除
おすすめ配合から選ぶ場合も固定レシピを解除
```

## これで防げること

```text
配合リストではAと表示
↓
この配合をセット
↓
配合画面ではBが表示される
```

というズレを防ぎます。

## レシピ重複チェック

```text
親ペア重複: 0
なし
```

## 変更していないもの

```text
配合レシピ内容
配合後Lv1仕様
保護ロック仕様
スカウト率
モンスター能力値
ステージ出現
```
