# v7.6.1 牧場2列表示改善

## 目的

仲間ページの牧場エリアが、横幅に余裕があるのに1列表示になっていたため、PC・横幅の広い画面で見やすくしました。

## 変更内容

```text
パーティと牧場のレイアウトを縦並びに変更
牧場カードを全幅表示に変更
牧場内のモンスターカードを広い画面で2列表示
スマホでは従来通り1列表示
```

## 表示仕様

```text
900px以上: 牧場2列表示
900px未満: 牧場1列表示
1120px以上: パーティ側も2列表示
```

## 変更していないもの

```text
パーティ最大数
牧場の中身
モンスター能力値
配合仕様
サイズ表示仕様
戦闘バランス
```

## 検証

```text
親ペア重複: 0
存在しないモンスターID参照: 0
```
