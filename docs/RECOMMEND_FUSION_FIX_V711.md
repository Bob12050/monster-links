# v7.1.1 おすすめ配合の結果ズレ修正

## 不具合

おすすめ配合を選択したときに、おすすめ欄に表示されていた配合結果と、選択後のプレビュー結果が違うことがありました。

## 原因

通常配合の一部結果がランダム抽選になっていたためです。

```text
おすすめ表示時に結果Aを計算
↓
この2体を選ぶ
↓
同じ親2体で再計算
↓
ランダムで結果Bになる
```

このため、表示と選択後の結果がズレていました。

## 修正内容

通常配合のランダム抽選をやめ、同じ親IDの組み合わせなら必ず同じ結果になるようにしました。

```text
同じ親2体
↓
いつ見ても同じ配合結果
↓
おすすめ表示・プレビュー・実際の配合結果が一致
```

## 変更していないもの

```text
配合レシピ内容
配合後Lv1仕様
スカウト率調整
モンスター能力値
ステージ出現
```
