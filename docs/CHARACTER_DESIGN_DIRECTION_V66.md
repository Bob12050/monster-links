# v6.6 キャラデザイン方針

## 目指す方向

Monster Linksは、将来的にドラクエモンスターズ風の育成・配合体験に寄せます。
ただし、名前・見た目・世界観はオリジナルで作ります。

## デザインの方向性

```text
低ランク: かわいい・分かりやすい・親しみやすい
中ランク: 少し強そう・属性が分かる
高ランク: 派手・威厳・シルエットが強い
Sランク: ボス級・特別感
SS/SSS: 神話級・超大型・隠し要素
```

## 1枠/2枠/3枠の見た目

```text
1枠: 小型〜中型、通常モンスター
2枠: 大型、横幅や迫力がある
3枠: 超大型、ボス級、画面映え重視
```

## 属性ごとの見た目メモ

```text
fire: 赤・橙・炎・角・爪
water: 青・水晶・ヒレ・泡
nature: 緑・葉・花・苔
stone: 岩・鉱石・重厚感
machine: 金属・歯車・発光ライン
light: 白・金・羽・星
dark: 黒・紫・影・魔眼
dragon: 角・翼・鱗・長いシルエット
wing: 羽・軽さ・素早さ
slime: 丸い・柔らかい・表情
beast: 毛並み・牙・爪・俊敏さ
```

## 画像生成時の基本プロンプト方針

```text
透過PNG
モンスター単体
ゲーム用キャラクター
正面または斜め前
全身が見える
背景なし
アイコンとして見やすい
```

## 避けること

```text
既存ゲームのキャラに似すぎる
名前や見た目をそのままコピーする
背景付き画像
細部が細かすぎてスマホで潰れる
低ランクなのに強そうすぎる
Sランクなのに地味すぎる
```
