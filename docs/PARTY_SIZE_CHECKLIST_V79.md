# v8.0 動作確認チェックリスト

## 通常確認

- [ ] タイトル画面が `v8.0` 表示になっている
- [ ] 拠点画面の説明がパーティ枠管理の内容になっている
- [ ] 仲間画面にパーティ枠ゲージが表示される
- [ ] パーティが1枠モンスター3体の時、`3/3枠` になる
- [ ] 2枠モンスター + 1枠モンスターの時、`3/3枠` になる
- [ ] 2枠モンスター + 2枠モンスターにしようとすると、後から入れる方が拒否される

## 詳細モーダル確認

- [ ] パーティのモンスターをタップすると詳細モーダルが開く
- [ ] 牧場のモンスターをタップすると詳細モーダルが開く
- [ ] 詳細モーダルに対象モンスターの枠数が表示される
- [ ] 枠不足の牧場モンスターは「パーティへ」ボタンが無効になる

## 配合・スカウト確認

- [ ] 配合後の子は今まで通りLv1で生まれる
- [ ] 引き継ぎ技は今まで通り最大2つ選べる
- [ ] パーティ枠に空きがあれば子がパーティへ入る
- [ ] パーティ枠に空きがなければ子が牧場へ入る
- [ ] スカウトした仲間も、枠が足りなければ牧場へ入る
