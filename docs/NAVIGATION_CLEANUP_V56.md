# v5.6 メニュー・導線の最終整理

## 今回の目的

機能が増えてきたため、メニューと導線を整理しました。

## 変更内容

```text
メニューの分類を「冒険・育成」「記録・挑戦」「管理」に整理
配合は直接メニューよりも仲間画面から進む導線を優先
ホームのクイック操作の重複を整理
「次にやること」の最終案内を仲間画面へ寄せる
下部タブの考え方を維持しつつ、メニューの役割を補助機能寄りに変更
```

## 方針

```text
よく使う操作: 拠点 / 冒険 / 仲間
補助機能: メニュー
配合: 仲間画面から入る
設定・遊び方: メニュー
```

## 変更していないもの

```text
戦闘バランス
配合内容
アイテム性能
ステージ構成
画像アセット
```
