# v7.2.5 引き継ぎ技の選び直し修正

## 目的

v7.2.4でも引き継ぎ技を選び直しにくい状態が残っていたため、操作方式を変更しました。

## 修正内容

```text
2つ選択済みの状態で別の技を押せるように変更
3つ目を押した場合、古い選択を1つ外して新しい技を追加
選択済みの技を押した場合は従来通りOFF
選択中の技が見やすいように表示を強調
```

## 新しい操作

```text
技A・技Bが選択中
↓
技Cを押す
↓
技Aが外れて、技B・技Cになる
```

これで、毎回手動で外してから選ぶ必要がなくなります。

