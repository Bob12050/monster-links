# v5.9.3 配合リストから配合セット機能追加

## 今回の目的

配合リストを見ながら、素材モンスターを手動で探して選ぶ手間を減らしました。

## 追加内容

```text
配合リストの各レシピに「この配合をセット」ボタンを追加
手持ちに親モンスターがいる場合、ワンタップで親2体を選択
レベル条件があるレア配合は「Lv条件確認」と表示
素材が足りないレシピは「素材不足」と表示して無効化
```

## 使い方

```text
仲間
↓
配合へ
↓
配合リストを開く
↓
この配合をセット
↓
上部の配合プレビューを確認
↓
この2体で配合
```
