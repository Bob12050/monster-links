# GitHub反映前チェックリスト v6.3

## PCで反映する手順

```text
1. v6.3 分割版ZIPをダウンロード
2. ZIPを解凍
3. 中身だけをGitHub用フォルダに上書き
4. GitHub Desktopで変更内容を確認
5. Commit
6. Push origin
7. GitHub Pagesの公開URLで確認
```


## 特に確認する操作

```text
配合リストから「この配合をセット」
開発者モードON時のパスワード
全キャラ入手
修練の書のまとめ買い
スカウト笛のまとめ買い
スカウト笛ストックの消費
アクセサリーが1個購入になっているか
戦闘ダメージが重すぎないか
```

## 友達に送る前に確認

```text
開発者モードがOFFになっている
タイトル画面が表示される
遊び方ページが開ける
序盤の戦闘が遊べる
配合リストが開ける
スマホで表示崩れがない
```
