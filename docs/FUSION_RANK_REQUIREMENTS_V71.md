# v7.1 配合ランク条件表

## 目的

Fランクから段階的に上位ランクを目指せる楽しさは残しつつ、短時間でSランクまで到達しすぎる問題を抑えます。

## 条件表

| 作成ランク | 親平均Lv条件 | 親ランク条件 |
|---|---:|---|
| F | なし | なし |
| E | 3以上 | なし |
| D | 6以上 | なし |
| C | 10以上 | なし |
| B | 15以上 | 親のどちらかC以上 |
| A | 22以上 | 親のどちらかB以上 |
| S | 32以上 | 親のどちらかA以上、両方B以上 |

## 狙い

```text
FからSを目指せる
ただし育成と再配合が必要
配合した瞬間に即戦力になりすぎない
高ランクほど素材作りが重要になる
```

## 注意

既存レア配合に設定されている minAvg より、この共通条件の方が高い場合は共通条件が優先されます。
