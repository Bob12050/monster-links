# Monster Links 機械系整理資料

更新日: 2026-06-09  
対象バージョン: v8.6-A.2  
資料種別: アート・系統整理メモ  
ゲーム反映状態: v8.6-A.3で既存ID分を追加反映済み

この文書は、Monster Links の「機械系」モンスターの系統、役割、アート方針、現在のデザイン承認状況を整理するための資料です。

ここでの「採用候補」「承認」は、白背景確認用デザインの方向性承認を意味します。  
透明PNG化、画像ファイル差し替え、表示名変更、ゲーム実装は、ユーザー確認後の別作業として扱います。  
特にランクについては、現時点では「アート上の格・将来候補」として整理し、ゲーム内の実データはまだ変更しません。

---

## 1. 基本方針

機械系は、歯車、金属装甲、蒸気、古代機械、発光コア、ドローン、機械獣、巨大兵器を表現する系統です。

共通方針:

```text
銀、黒鉄、ガンメタル、真鍮、金、青緑の発光を主軸にする
歯車、ボルト、リベット、装甲板、関節、発光コアを使う
低ランクは小型で親しみやすく、中位は兵器・昆虫・獣型へ広げる
上位は古代兵器・守護者・巨大機械として格を出す
スマホ画面でもシルエットが分かる
既存の維持対象であるギガントコア、オメガフレームの格を壊さない
```

機械系は、他系統よりも見た目が良い既存モンスターが多いため、  
すでに完成度が高いものは無理に差し替えず、未整備・方向性未確定のものだけ再設計します。

## 6. ランク整理案

現時点では実データは変更しませんが、アート上の格は以下で整理します。

```text
D: ギアビット、ギアキャット、ギアぷるミン
C: スチールバグ、アイアンマンティス、サンダードローン
B: フォージゴーレム

S相当: スカイウォーデン、オメガフレーム
SS候補: アークマシン
```

注意:

```text
スカイウォーデンとアークマシンの実ランク変更はまだしない
SSランク導入時にアークマシンを再検討する
スカイウォーデンはS化、または上位派生追加も検討可能
```

## 7. 実装時の注意

```text
確認済みデザインのみ透過PNG化する
未承認アートをゲームに正式反映しない
画像更新と能力・配合・セーブ形式変更を混ぜない
既存IDを安易に変更しない
表示名変更だけで済む場合はIDを維持する
画像は assets/images/monsters/{id}.png に配置する
旧画像は必要に応じて legacy フォルダへ退避する
実装後は tools/validate.mjs と tools/art-audit.mjs を実行する
```

ランク変更を行う場合は、画像実装とは別作業にします。

```text
rank変更
配合レシピ変更
ステージ出現変更
能力値調整
図鑑・研究依頼調整
```

これらは後でまとめて検討します。
