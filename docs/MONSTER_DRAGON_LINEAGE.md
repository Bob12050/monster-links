# Monster Links ドラゴン系整理資料

更新日: 2026-06-10  
対象バージョン: v8.6-A.4  
資料種別: アート・系統整理メモ  
ゲーム反映状態: 採用済み7体をv8.6-A.4で実装済み

## v8.6-A.4 実装メモ

- `cindrake`, `crystagon`, `volcazard`, `venomhydra`, `prismdragon`, `heavenscale`, `zenithdragon` の採用済みアートを既存IDのまま実装しました。
- 全7体を透過1024×1024 PNGへ整え、`heavenscale` と `zenithdragon` も正式PNGへ切り替えました。
- ヴェノムハイドラなどの「アート上の格」は資料上の整理であり、ゲーム内の実データの `rank` は変更していません。
- ランク、能力値、配合、セーブ形式は変更していません。

この文書は、Monster Links の「ドラゴン系」モンスターの系統、役割、アート方針、現在のデザイン承認状況を整理するための資料です。

ここでの「採用候補」「承認」は、白背景確認用デザインの方向性承認を意味します。  
透明PNG化、画像ファイル差し替え、ランク変更、配合変更、ゲーム実装は、ユーザー確認後の別作業として扱います。

特にランクについては、現時点では「アート上の格・将来候補」として整理し、ゲーム内の実データはまだ変更しません。

---

## 1. 基本方針

ドラゴン系は、Monster Links の中でも上位感・成長感・最終到達点を強く出せる系統です。

共通方針:

```text
翼、角、爪、牙、尾、鱗でドラゴンらしい強さを出す
低ランクは小型で親しみやすく、中位から体格と装飾を増やす
高ランクは体格、翼、角、鱗、宝玉、発光、神聖感で格を出す
火、氷、結晶、毒、深海、天界、虹光、混沌など属性差を明確にする
スマホ画面でもシルエットが分かる
背景演出ではなく、モンスター単体の造形で強さを出す
白背景・単体・全身中央配置を基本にする
```

ドラゴン系はSランクが多いため、Sランク同士の差別化が重要です。

## 6. ランク整理案

現時点では実データは変更しませんが、アート上の格は以下で整理します。

```text
D: シンドレイク
C: クリスタゴン
B: ヴォルカザード
A: フロストリヴァイア
S相当: ヴェノムハイドラ
S: アビスリヴァイア、プリズムドラゴン、ヘヴンスケイル、ゼニスドラゴン、カオスワーム
```

注意:

```text
ヴェノムハイドラの実ランク変更はまだしない
ヴォルカザードは見た目が強いため実装前に最終確認する
Sランクの数が多いので、配合難度・出現時期・能力値で差別化する
```

## 7. 実装時の注意

```text
確認済みデザインのみ透過PNG化する
未承認アートをゲームに正式反映しない
画像更新と能力・配合・セーブ形式変更を混ぜない
既存IDを安易に変更しない
画像は assets/images/monsters/{id}.png に配置する
旧画像は必要に応じて legacy フォルダへ退避する
実装後は tools/validate.mjs と tools/art-audit.mjs を実行する
```

ランク変更を行う場合は、画像実装とは別作業にします。

```text
rank変更
配合レシピ変更
ステージ出現変更
能力値調整
図鑑・研究依頼調整
```

これらは後でまとめて検討します。
