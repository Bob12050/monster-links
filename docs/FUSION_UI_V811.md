# v8.1.1 配合検索・確認UI

## 配合リスト検索

```text
結果モンスター名
親素材モンスター名
モンスターID
レシピの説明
```

を検索対象にしています。

## 複合フィルター

```text
状態:
すべて
作成可能
素材あり・条件未達
未発見

結果サイズ:
すべて
1枠
2枠
3枠
```

名前検索・状態・結果サイズは同時に利用できます。

## 配合確認モーダル

```text
消費する親2体
誕生する子
親と子のサイズ
配合種類
パーティ加入または牧場送り
引き継ぎ技
親が装備しているアクセサリーの返却
親2体がいなくなる警告
```

確定ボタンを押した時点でも親・保護状態・配合結果を再確認します。
確認中に内容が変わった場合は実行せず、もう一度確認を求めます。
