# v7.2.7 現在の配合仕様まとめ

## 基本仕様

```text
親2体を選んで子モンスターを作る
親2体は配合後にいなくなる
子は必ずLv1で生まれる
親の個体値・能力ボーナス・一部スキルを引き継ぐ
```

## 保護ロック

```text
保護中の仲間は配合素材にできない
配合リストやおすすめ配合でも素材候補から除外
```

## 配合リストから選んだ場合

```text
配合リストから選んだレシピ結果を固定
プレビューと実際の配合結果が一致
手動で親を選び直すと固定レシピは解除
```

## 高ランク配合条件

```text
Bランク作成: 親平均Lv15以上 + 親のどちらかC以上
Aランク作成: 親平均Lv22以上 + 親のどちらかB以上
Sランク作成: 親平均Lv32以上 + 親のどちらかA以上 + 両方B以上
```

## 引き継ぎ技

```text
最大2つまで選択
技選択をクリア可能
2つ選択中に別の技を押すと、その技だけ選択した状態にリセット
0個のまま配合した場合は、実行時に候補から自動補完
```
