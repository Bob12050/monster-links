# v7.0.1 図鑑が開けない不具合修正

## 原因

図鑑カードでモンスターの習得スキル名を表示する時に、スキル定義が存在しないIDを直接参照すると、画面描画が止まる可能性がありました。

## 修正内容

```text
図鑑のスキル表示を安全化
未定義スキルがあっても「未定義:<id>」として表示
属性表示も安全化
フィルター側の属性名参照も安全化
必要に応じて不足スキルのフォールバックを追加
```

## 検出された未定義スキル参照

```text
- shellgolem: guard
```

## 修正後の未定義スキル参照

```text
なし
```
