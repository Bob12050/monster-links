# v6.9 追加モンスター第1弾 能力値方針

## 方針

追加20体は、既存ランクの強さに近づけて能力値を設定しました。

## ランクごとの考え方

```text
F: 序盤で使いやすいが低ステータス
E: 序盤〜洞くつあたりの通常戦力
D: 中盤入口の個性枠
C: 中盤の主力候補
B: 後半入口・配合で狙う強め枠
A: 終盤手前の上位戦力
S: 特別感のあるレア枠
```

## sizeについて

forgegolem と titanplim は `size:2` を持たせていますが、現時点では表示・パーティ制限には使っていません。

```text
forgegolem: size 2候補
titanplim: size 2候補
```

正式な2枠/3枠システムは将来実装します。

## 注意

今回の追加で、開発者モードの「全キャラ入手」にも新20体が含まれるようになります。
