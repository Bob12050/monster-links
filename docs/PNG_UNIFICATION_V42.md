# v4.2 全モンスターPNG統一

## 目的

```text
透過問題の再発を防ぐ
画像運用を統一する
将来的な差し替えを簡単にする
```

## 今回の内容

- 既に本格PNGアート化済みの11体はそのまま維持
- 残りのモンスターSVGを透過PNGへ変換
- `js/core/monsters.js` の全モンスター画像参照を `.png` に統一

## 変換数

```text
新規SVG→PNG変換数: 44
既存本格PNGアート: 11体
```

## これでできること

- すべてのモンスターがPNGとして扱われる
- 本格アートを追加したい時は同名PNGで上書きするだけで良い
- SVGとPNGが混在して見た目が不安定になるのを防げる
