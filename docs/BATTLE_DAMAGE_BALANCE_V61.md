# v6.1 戦闘ダメージ全体の軽減・技の役割強化

## 今回の目的

友達レビューで「敵も味方もダメージ量が大きい」と指摘があったため、戦闘全体のダメージを少し抑えました。

## 調整方針

```text
敵だけ弱体化しない
味方だけ弱体化しない
敵味方どちらのダメージも少し下げる
通常攻撃だけで終わりにくくする
弱点を突いた技の価値は残す
防御と回復の価値を少し上げる
```

## 追加したバランス値

```text
playerDamageMultiplier: 0.82
enemyDamageMultiplier: 0.82
normalAttackMultiplier: 0.88
skillDamageMultiplier: 0.96
weaknessMultiplierBonus: 0.08
guardMultiplier: 0.38
healMultiplier: 1.12
```

## 主な変更

```text
与ダメージを全体的に軽減
被ダメージを全体的に軽減
通常攻撃を少し控えめに
技ダメージは少し抑えつつ、通常攻撃より価値が残るように調整
防御時の軽減率を強化
回復技の回復量を少し増加
防御力の影響を少し強める
```

## 変更していないもの

```text
敵ステータス
味方ステータス
技の消費MP
技の基本威力データ
経験値
ゴールド
スカウト率
配合
```

## 確認ポイント

```text
一撃で倒す/倒される場面が減ったか
通常攻撃だけで終わりにくくなったか
弱点技を使う価値があるか
回復・防御の意味が少し出たか
戦闘が長すぎないか
```
