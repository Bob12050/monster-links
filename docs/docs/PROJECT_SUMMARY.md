# Monster Links プロジェクト概要

## 概要

モンスターを仲間にし、育成・配合・装備・ステージ攻略を進める、スマホ向けのソロRPGです。

## 現在できること

```text
ステージ探索
ボス戦
モンスターのスカウト
モンスター育成
配合
装備
アイテム
図鑑
任務
闘技場
複数セーブスロット
```
