# v5.9.1 開発者モード拡張：全キャラ入手追加

## 今回の目的

キャラデザ確認や配合確認をしやすくするため、開発者モードに「全キャラ入手」を追加しました。

## 追加内容

```text
開発者モードに「全キャラ入手」ボタンを追加
未所持モンスターを1体ずつ仲間に追加
既に持っている種族は重複追加しない
追加モンスターはランクに応じた確認用レベルで生成
```

## 使い方

```text
設定・セーブ
↓
開発者モード ON
↓
メニュー
↓
開発者モード
↓
全キャラ入手
```

## 注意点

```text
現在のセーブスロットに反映されます
確認用に別スロットへコピーしてから使うのがおすすめです
友達に遊んでもらう時は開発者モードOFF推奨です
```
