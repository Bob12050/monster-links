# v6.2.1 まとめ買い対象修正

## 修正内容

v6.2ではアクセサリー側にまとめ買いを追加しましたが、レビューで求められていたのは主に以下でした。

```text
キャンプセット
修練の書
スカウト笛
```

そのため、v6.2.1では便利アイテム側にまとめ買いを追加しました。

## 追加内容

```text
修練の書: 1回 / 5回 / 10回 / 買えるだけ
スカウト笛: 1回 / 5回 / 10回 / 買えるだけ
```

## キャンプセットについて

キャンプセットはその場で全回復する効果なので、複数回まとめて買っても意味が薄いです。
そのため、キャンプセットは従来通り1回購入のままにしています。

## スカウト笛の変更

スカウト笛は購入回数分だけストックされるようにしました。

```text
スカウト笛を5回分購入
↓
次の戦闘から5戦分、スカウト率アップ
```
