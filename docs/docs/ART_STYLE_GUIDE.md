# アート方針書 v3.9

## 目的

モンスターリンクスを、試作ゲームから「ちゃんとしたゲーム」に近づけるため、画像の絵柄・サイズ・命名ルールを統一します。

## モンスターアート方針

```text
ファンタジーRPGらしいデフォルメ調
かわいさはあるが、子供っぽすぎない
スマホ画面でも見分けやすい
シルエット重視
属性や系統が色で分かる
太めで見やすい輪郭
正面〜やや斜め向き
背景透過
```

## モンスター画像仕様

```text
形式: PNG
サイズ: 1024 x 1024
背景: 透過
構図: 中央配置
余白: 画像端から10〜15%程度
用途: 戦闘、図鑑、仲間一覧、配合画面で共通使用
```

## ステージ背景仕様

```text
形式: PNG
サイズ: 1600 x 900 または 1280 x 720
背景: 透過なし
構図: UIが重なる前提で、中央や下部をうるさくしすぎない
用途: ステージ選択、戦闘背景
```

## アイテム画像仕様

```text
形式: PNG
サイズ: 512 x 512
背景: 透過
構図: 中央配置
条件: 小さく表示しても判別できること
```

## 属性カラー目安

```text
slime   明るい青・水色
nature  緑・黄緑・茶
fire    赤・橙・黄
water   青・紺・シアン
stone   灰・茶・鈍い青
light   白・金・薄青
dark    紫・黒・深紅
machine 銀・鉄色・青白
dragon  濃色＋属性アクセント
wing    白・水色・薄紫
beast   茶・黄・赤茶
```

## 統一感を崩さないためのルール

```text
画像のタッチを急にリアル寄りにしすぎない
モンスターごとの解像度や塗りの細かさを揃える
背景はモンスターより目立たせすぎない
アイテムは小さい表示での読みやすさを優先
PNG名はゲーム内IDと一致させる
```
