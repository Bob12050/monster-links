# 画像生成プロンプトテンプレート

## モンスター用テンプレート

```text
背景透過PNG用のファンタジーRPGモンスターイラスト。
キャラクター名: {name}
系統: {type}
ランク: {rank}
特徴: {feature}
かわいさはあるが子供っぽすぎない、スマホゲーム向け、太めの輪郭、はっきりしたシルエット、正面から少し斜め向き、中央配置、1024x1024、背景透過、統一感のあるデフォルメRPGアート。
```

## ステージ背景用テンプレート

```text
ファンタジーRPGのステージ背景。
ステージ名: {name}
雰囲気: {mood}
UIの後ろに敷いても見やすい、情報量は多すぎない、奥行きのある背景、横長構図、1600x900、PNG、背景透過なし。
```

## アイテム用テンプレート

```text
ファンタジーRPGのアイテムアイコン。
アイテム名: {name}
種類: {type}
小さく表示しても分かりやすい、中央配置、太めのシルエット、少し立体感、512x512、背景透過PNG。
```

## 注意

生成した画像は、必ず以下の命名で保存してください。

```text
assets/images/monsters/{id}.png
assets/images/stages/{id}.png
assets/images/items/{id}.png
```
