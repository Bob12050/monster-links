# 友達に遊んでもらう時の確認リスト

## 送る前に確認

```text
タイトル画面が表示される
続きから遊ぶボタンが押せる
遊び方ページが開ける
冒険ステージが開ける
戦闘が開始できる
勝利後の報酬画面が表示される
仲間画面が開ける
仲間画面から配合へ行ける
配合リストが開閉できる
メニューが開ける
設定・セーブが開ける
```

## 友達に聞くと良いこと

```text
最初に何をすればいいか分かったか
画面がごちゃごちゃしていないか
ボタンは押しやすいか
戦闘のテンポは悪くないか
序盤の難易度は高すぎないか
配合の場所は分かりやすいか
スマホで表示崩れがないか
```

## 友達に送る時の一言

```text
スマホでも遊べるモンスター育成RPGを作ってます。
最初は「遊び方」を見てから、冒険で敵を倒して仲間を増やしてみてください。
変なところがあったらスクショ送ってくれると助かります。
```
