# v6.2時点のコード整理判断

## 結論

v6.2では、大規模なコード整理はまだ行いません。

## 理由

直近で以下の重要機能を追加・調整しています。

```text
開発者モード
開発者モードのパスワード
全キャラ入手
配合リストから配合セット
戦闘ダメージ調整
まとめ買い
```

この直後にファイル移動や大規模リファクタリングをすると、不具合原因の切り分けが難しくなります。

## 今やった整理

```text
まとめ買いの処理は shop.js に集約
表示は shopView.js に限定
変更内容は docs に記録
```

## 本格整理の候補タイミング

```text
v6.4 または v6.5
```

その時に見る候補:

```text
READMEの短縮
古いdocsの棚卸し
CSS末尾の重複整理
shop/fusion/devtools周りの関数整理
```

## まだやらない整理

```text
JSファイル大移動
CSS分割
データ構造の全面変更
assetsフォルダ変更
古いdocsの大量削除
```
