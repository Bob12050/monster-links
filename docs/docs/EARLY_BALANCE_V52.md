# v5.2 序盤難易度の軽調整

## 今回の目的

友達や初見プレイヤーが序盤で詰まりにくくなるように、序盤3ステージだけ軽く調整しました。

## 調整対象

```text
はじまり草原
こだま洞くつ
しずく湖畔
基本ショップアクセサリー
```

## ステージ調整

### はじまり草原

```text
ゴールド: [8,14] → [11,18]
経験値: [12,20] → [16,26]
スカウト率: 34 → 42
ボス報酬: gold 55 → 70 / exp 90 → 115 / scout 12 → 14
```

### こだま洞くつ

```text
ゴールド: [16,28] → [20,34]
経験値: [22,38] → [28,46]
スカウト率: 28 → 34
ボス気配必要数: 3 → 2
ボス報酬: gold 95 → 120 / exp 150 → 190 / scout 9 → 11
```

### しずく湖畔

```text
ゴールド: [26,42] → [32,52]
経験値: [38,62] → [48,76]
スカウト率: 23 → 28
ボス報酬: gold 150 → 185 / exp 245 → 310 / scout 7 → 8
```

## ショップ価格調整

```text
ちからのリング: 120 → 95
まもりの石: 120 → 95
はやての羽: 120 → 95
まどうの首飾り: 140 → 110
命のしずく: 160 → 125
マナシェル: 150 → 120
```

## 変更しなかったもの

```text
敵ステータス
戦闘計算式
後半ステージ
ステージギミック
ボスの強さ
```

## 方針

難易度を大きく下げるのではなく、序盤の育成・装備・スカウトのテンポを良くする調整です。
