# v8.1 サイズ別配合ルート

## ルール

```text
通常配合:
子のサイズは親2体の合計サイズ以下

2枠結果:
親平均Lv8以上

3枠結果:
親平均Lv32以上
親2体の合計サイズ3枠以上
Sランク作成条件も必要
```

固定レシピは従来の組み合わせを維持しています。
既存の高ランク条件やレシピ固有の平均Lv条件が高い場合は、そちらを優先します。

## UI

```text
親サイズ合計 → 子サイズをプレビュー表示
配合後にパーティへ入るか牧場へ送られるか表示
3枠結果は単独編成になることを警告
配合リストを1枠・2枠・3枠で絞り込み
```

## セーブ互換性

保存データ構造は変更していません。
既存セーブはそのまま利用できます。
