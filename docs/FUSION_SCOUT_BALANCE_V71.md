# v7.1 配合後Lv1化・スカウト率調整・高ランク到達抑制

## 今回の目的

友達レビューで「10〜20分でSランクが作れてしまう」問題が見つかったため、配合とスカウトの進行速度を調整しました。

## 配合後Lv1化

```text
配合後の子モンスターは必ずLv1で誕生
親の個体値・能力ボーナス・一部スキルは引き継ぎ
親2体は消える
装備はバッグに戻る
```

## ランク別の親平均Lv条件

```text
Eランク作成: 親平均Lv3以上
Dランク作成: 親平均Lv6以上
Cランク作成: 親平均Lv10以上
Bランク作成: 親平均Lv15以上
Aランク作成: 親平均Lv22以上
Sランク作成: 親平均Lv32以上
```

## 高ランク配合の親ランク条件

```text
Bランク作成: 親のどちらかがCランク以上
Aランク作成: 親のどちらかがBランク以上
Sランク作成: 親のどちらかがAランク以上、かつ親2体がどちらもBランク以上
```

## スカウト率調整

```text
全体のスカウト補正を +6 から +2 に低下
ランクによるスカウト難度を少し強化
A/Sランクは追加ペナルティ
スカウト失敗ごとに成功率 -10%
4回失敗すると、その戦闘ではスカウト不可
```

## 変更していないもの

```text
モンスター能力値
ステージ出現
配合レシピ内容
ドロップ
戦闘ダメージ
```
