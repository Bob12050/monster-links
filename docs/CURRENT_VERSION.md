## 今回の目的

```text
ステージ一覧を大陸地図へ変更し、地域を進んでいる感覚を強くする。
情報量を減らさず、現在地と次の目標を見つけやすくする。
```

## 今回の変更

```text
GAME_VERSIONを8.5.1へ更新
13地域を曲線ルートで結んだ大陸ワールドマップを追加
地点ピンに地域番号、名称、アイコン、進行状態を表示
解放済み、ボス挑戦可能、制覇済み、未解放を色分け
最新解放地域を初期選択
ロック地域も地図上に表示し、次の目的地を確認可能
従来の全カード一覧を、地図と選択地域詳細の2段構成へ変更
推奨Lv、敵Lv、危険度、ボス進行を地域ステータスへ集約
属性傾向、出現モンスター、主な報酬を詳細パネルへ表示
ボス情報と攻略ヒントを選択地域ごとに表示
スマホでは地図部分だけを横スクロール
選択地点を地図中央付近へ自動スクロール
演出軽減設定では地図ピンのアニメーションを停止
13地点生成、ロック表示、地域選択、不正ID耐性を回帰テストへ追加
戦闘開始、ボス解放、報酬計算、ステージデータは変更なし
セーブ形式番号は変更なし
```
