## 今回の目的

```text
戦闘画面の情報は揃っていたが、敵味方が戦っている臨場感と
コマンドごとの役割、控えメンバーの状態が分かりにくかった。
```

## 今回の変更

```text
GAME_VERSIONを8.2.1へ更新
敵を右上、味方を左下に配置する対面戦場へ変更
敵味方のHP・MP・ランク・属性を専用HUDに表示
控えメンバーと残りHPを戦闘画面に表示
最新行動、行動待機状態、属性相性を常時表示
6種類のコマンドを用途説明付きの操作盤へ変更
特技選択に属性、消費MP、敵との相性を表示
交代選択にモンスター画像、HP、現在の状態を表示
攻撃元の踏み込み、被ダメージ、回復、防御、スカウト失敗演出を強化
撃破・全滅・自動交代の前に短い演出時間を追加
PC・スマホ表示を調整
戦闘計算、敵AI、報酬、セーブ形式は変更なし
```
