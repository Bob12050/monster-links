# 現在の安定版

```text
v8.4.1 通常攻撃オート戦闘
```

## 今回の目的

```text
戦闘速度改善と組み合わせて通常戦闘の周回操作を減らす。
回復やスカウトを勝手に判断せず、安全に通常攻撃だけを自動化する。
```

## 今回の変更

```text
GAME_VERSIONを8.4.1へ更新
戦闘画面にAUTO ON/OFFボタンを追加
オート中はプレイヤーターンごとに通常攻撃を自動選択
オート状態を戦闘メッセージ欄へ表示
特技、スカウト、防御、交代、逃走を選ぶと即時解除
特技・交代メニューを開いた時点でも解除
HP25%以下ではオートを開始せず、実行中も安全停止
戦闘不能、敵撃破、勝敗決定時に停止
通常戦・ボス戦・闘技場の開始時は必ずOFF
闘技場の次ラウンド開始時もOFF
予約タイマーが古い戦闘や行動中の戦闘へ作用しないよう防御
手動攻撃と予約攻撃が重なった場合も戦闘ロックで二重行動を防止
オート状態はセーブデータに保存しない
回復、特技選択、スカウト判断、戦闘計算、敵AIは変更なし
```
