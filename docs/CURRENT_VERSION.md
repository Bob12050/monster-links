## 修正した問題

```text
パーティには最低1体必要なため、現在の仲間を先に全員牧場へ戻せない。
その結果、3枠モンスターを牧場からパーティへ入れられない。
```

## 今回の変更

```text
GAME_VERSIONを8.1.2へ更新
枠不足時のパーティへボタンを交換ボタンへ変更
牧場へ戻すパーティメンバーを複数選択
交換後の使用枠と編成を事前表示
必要枠を満たす組み合わせを最初から自動選択
1枠3体から3枠1体への一括交換に対応
1枠2体から2枠1体への交換に対応
交換は最後に一括適用し、途中でパーティ0体にしない
選択不足や対象不在では編成データを変更しない
```
