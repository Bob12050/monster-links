## 今回の目的

```text
欲しいモンスターを見つけた後に、
必要素材の確認・育成・配合まで迷わず進めるようにする。
図鑑、拠点、配合所を一つの目標でつなぐ。
```

## 今回の変更

```text
GAME_VERSIONを8.3.2へ更新
発見済み図鑑カードに配合目標の星ボタンを追加
図鑑詳細から配合目標を登録・解除可能
配合目標を最大3体保存
目標ごとに最適な固定配合ルートを自動選択
未保護素材の所持数、必要数、最高Lv、保護数を表示
素材不足、育成条件あり、今すぐ配合可能を色分け
拠点に最優先の配合目標を表示
配合所に全目標と素材進捗を表示
作成可能な目標から親2体を自動セット
目標の優先順位変更と解除に対応
未発見素材の名前は従来どおり非表示
旧セーブには空のfusionGoalsを自動補完
配合結果、戦闘、パーティ枠の仕様は変更なし
```
