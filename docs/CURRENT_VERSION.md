
## 今回の目的

```text
配合リストだけでは、欲しいモンスターを図鑑で見つけた後に
必要な親を探す流れが分断されていた。
図鑑を育成目標と配合計画の入口にする。
```

## 今回の変更

```text
GAME_VERSIONを8.3へ更新
発見済み図鑑カードをタップ可能に変更
モンスター画像・図鑑状態・所持数を表示する詳細モーダルを追加
対象モンスターを作れる固定配合ルートを表示
対象モンスターを素材に使う固定配合ルートを逆引き表示
素材所持数、未保護数、保護数、最高Lvを表示
作成可能、条件未達、素材不足、未発見を色分け
素材と条件を満たすレシピを配合画面へ直接セット
未発見の親と結果は名前・画像・レシピキーを表示しない
PC・スマホ表示を調整
配合結果、条件、セーブ形式は変更なし
```
