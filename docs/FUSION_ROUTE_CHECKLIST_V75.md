# v7.5 配合ルート表示確認チェックリスト

## 確認したいこと

```text
配合リスト上部にサマリーが表示される
今すぐ作れる候補が表示される
各レシピに素材所持数が表示される
各レシピに素材最高Lvが表示される
保護中素材がある場合、保護数が表示される
作成可能/条件未達/素材不足が分かる
```

## 確認手順

```text
仲間
↓
配合
↓
配合リスト
↓
表示を確認
```

## 開発者モード確認

```text
開発者モードON
↓
全キャラ入手
↓
配合リスト
↓
作成可能候補が増えるか確認
```

## 注意

保護ロック中の仲間は素材数に含めません。保護中の数は別表示します。
