# v7.1.6 セーブデータバックアップ/復元

## 目的

アップデートや配合レシピ追加の前に、現在のセーブデータを安全に退避できるようにしました。

## 追加内容

```text
現在スロットのバックアップ書き出し
バックアップ文字列のコピー
バックアップ文字列から現在スロットへ復元
復元前の確認画面
復元時の安全化
```

## 使い方：バックアップ

```text
設定・セーブ
↓
データ管理
↓
バックアップを書き出し
↓
バックアップをコピー
↓
メモ帳などに保存
```

## 使い方：復元

```text
設定・セーブ
↓
データ管理
↓
バックアップから復元
↓
バックアップ文字列を貼り付け
↓
現在スロットへ復元
```

## 注意

```text
復元すると現在のスロットが上書きされます
心配な場合は先に現在データを書き出してください
バックアップ文字列は長文です
SNSよりメモアプリやテキストファイル保存がおすすめです
```

## 変更していないもの

```text
配合仕様
スカウト仕様
モンスター能力値
ステージ出現
配合レシピ
```
