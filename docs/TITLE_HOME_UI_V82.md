# v8.2 タイトル・拠点UI

## 目的

タイトルと拠点を、管理画面ではなくRPGの世界に入るための画面へ近づける。

## タイトル画面

- 前回の冒険地を背景に使用
- 現在のパーティを立ち絵で表示
- 現在のリーダーと前回の冒険地を表示
- 「冒険を再開」を最優先の操作として配置
- 所持金、パーティ枠、図鑑、任務の記録を表示

## 拠点画面

- 冒険門
- モンスター牧場
- 配合研究所
- 闘技場
- 任務掲示板
- 旅人の商店

各施設へ1回の操作で移動できる。次の目標は「冒険案内」として拠点上部に表示する。

## 互換性

セーブデータの項目、保存キー、保存形式は変更していない。
既存の画面遷移関数を利用しているため、従来のセーブデータをそのまま使用できる。
