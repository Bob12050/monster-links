# Monster Links ぷる系整理資料

更新日: 2026-06-08  
対象バージョン: v8.6-A.1  
資料種別: アート・系統整理メモ  
ゲーム反映状態: v8.6-A.2で既存ID分のみ反映

この文書は、Monster Links の「ぷる系」モンスターの系統、名称、アート方針を整理するための資料です。  
現時点では、ゲームデータ・配合・能力・セーブ形式・画像ファイルへの正式反映は行っていません。  
透明PNG化とゲーム実装は、ユーザー確認後の別作業として扱います。

---

## 1. 基本方針

ぷる系は、Monster Links の代表的なスライム系統として扱います。

共通方針:

```text
丸いしずく型・ぷるぷるした体
小さめで読みやすい顔
太めの輪郭
スマホ画面でも判別しやすいシルエット
派生ごとに色・質感・装飾を変える
かわいいが、完全なマスコットだけにはしない
既存作品の直接模倣はしない
```

ぷる系の再設計では、まず白背景の確認用イラストで方向性を確認し、ユーザー承認後に透過PNG化とゲーム反映を行います。

## 4. ケイブぷるミンとストーンぷるミンの扱い

ケイブぷるミンとストーンぷるミンは役割が被りやすいため、現時点では両方を同時実装しません。

現在方針:

```text
ケイブぷるミンを洞窟・岩石派生として採用候補にする
ストーンぷるミンは保留または不採用
```

ケイブぷるミンは以下の要素を持つため、ストーンぷるミンの役割もある程度吸収できます。

```text
灰色のぷる本体
洞窟感
岩・小石
結晶
苔
鉱物の垂れ・湿った質感
```

## 5. ぷる系の家系図

```text
ぷるミン
├─ ポイズンぷる
├─ 雲ぷる
├─ ギアぷるミン
```

旧しずくぷるはアワマリンへ再設計し、水系の独立モンスターとして扱います。

## 6. アート再設計の優先順位

ゲーム反映を行う場合の優先順位案:

```text
1. ぷるミン
2. キングぷるミン
3. ポイズンぷる
4. 雲ぷる
5. ケイブぷるミン
6. ギアぷるミン
7. タイタンぷる
```

理由:

```text
基本種を最初に確定する
次に王様系と分かりやすい派生を固める
ケイブぷるミンで洞窟・岩石派生を補う
ギアぷるミンは既存ID gearslime の扱いを慎重に確認してから反映する
タイタンぷるは最上位なので、他のぷる系との整合を確認してから正式反映する
```

注: 上記の `gearslime` は既存IDであり、表示名を「ギアぷるミン」へ変更する場合も、ID変更は慎重に扱います。

## 7. 実装時の注意

この資料は現時点では設計メモです。実装時は以下を守ります。

```text
確認済みデザインのみ透過PNG化する
未承認アートをゲームに正式反映しない
画像更新と能力・配合・セーブ形式変更を混ぜない
既存IDを安易に変更しない
表示名変更だけで済む場合はIDを維持する
画像は assets/images/monsters/{id}.png に配置する
旧画像は必要に応じて legacy フォルダへ退避する
実装後は tools/validate.mjs と tools/art-audit.mjs を実行する
```
