# 4体配合 将来設計

## 目的

SS/SSSランク級の特別なモンスターを作るための、超将来向け配合システムです。

## 基本方針

4体を一度に選ぶのではなく、段階式にします。

```text
A + B → 素材1
C + D → 素材2
素材1 + 素材2 → 特別モンスター
```

画面上では「4体配合」として見せつつ、内部的には通常配合を拡張する形が安全です。

## 例

```text
炎竜系 + 雷竜系 → 天火竜
氷竜系 + 闇竜系 → 冥氷竜
天火竜 + 冥氷竜 → SSランク: 神滅竜
```

## 必要な表示

```text
必要素材4体
中間素材2体
最終結果
素材不足
Lv条件不足
作成ルート
今持っている素材
あと何が足りないか
```

## 実装時の注意

```text
レシピ管理が複雑になる
配合リストが長くなる
素材不足表示が重要
開発者モードで検証できる必要がある
```

## 実装タイミング

```text
v7.0 SSランク追加後
v7.5 サイズ制の内部対応後
v8.0 4体配合実装
```

## 最初に入れるなら

最初は4体配合を3〜5種類だけにします。

```text
SSドラゴン
SS機械
SS神獣
SS闇竜
SS万能サポート
```

大量追加は後回しにします。
