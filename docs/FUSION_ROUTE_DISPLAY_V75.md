# v7.5 配合ルート表示強化

## 目的

配合リストを「ただの一覧」ではなく、今の手持ちから何が作れるか分かる画面に強化しました。

## 追加内容

```text
素材の所持数を表示
素材の最高Lvを表示
保護ロック中の素材数を表示
作成可能 / 条件未達 / 素材不足 を表示
配合リスト上部にサマリーを表示
すぐ作れる候補を上部に表示
```

## 表示される状態

```text
作成可能: 素材も条件も満たしている
条件未達: 素材はあるがLvやランク条件が足りない
素材不足: 親モンスターが足りない
```

## 変更していないもの

```text
配合レシピ内容
配合条件
配合結果固定仕様
配合後Lv1仕様
引き継ぎ技仕様
保護ロック仕様
スカウト率
モンスター能力値
ドロップ
```

## 検証

```text
レシピ数: 72
親ペア重複: 0
存在しないモンスターID参照: 0
```
