# v7.2.7 配合まわり安定化・確認強化

## 目的

v7.1〜v7.2.6で配合関連の仕様を多く追加したため、配合画面の見やすさと誤操作防止を強化しました。

## 追加・修正内容

```text
配合プレビューに「配合リストから選択中」を明確表示
固定レシピ/通常配合の表示を分かりやすく変更
引き継ぎ技の現在選択中リストを表示
技選択クリアボタンの表示位置を整理
配合実行前の確認文に引き継ぎ技を表示
配合実行前の確認文に配合種類を表示
```

## 配合確認文で表示される内容

```text
親2体の名前とLv
配合結果
配合種類
引き継ぎ技
親2体が消えること
保護ロック推奨
```

## 変更していないもの

```text
配合レシピ内容
配合結果固定仕様
配合後Lv1仕様
高ランク配合条件
保護ロック仕様
スカウト率
モンスター能力値
```
