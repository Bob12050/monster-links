# v7.2.2 配合画面に入れない不具合修正

## 原因

v7.1.5で追加した保護ロックボタンの参照位置に問題がありました。

`monsterActions()` の中で `lockBtn` を使っていましたが、`lockBtn` が `monsterCard()` 側の中だけで作られていたため、配合画面の仲間カードを描画する時にエラーになっていました。

## 修正内容

```text
monsterActions() 内で lockBtn を作るように修正
配合画面の仲間カード描画を安全化
パーティ画面・牧場画面・配合画面で保護ボタンを表示できるように修正
```

## 変更していないもの

```text
配合レシピ
配合後Lv1仕様
保護ロック仕様
スカウト率
モンスター能力値
ステージ出現
```

## 確認項目

```text
仲間画面が開ける
配合画面が開ける
配合画面で保護中モンスターが選べない
配合リストが開ける
```
